Terraform + Ansible Infrastructure Automation

End-to-End DevOps Portfolio Project

Project Overview

This project demonstrates a real-world infrastructure automation setup using Terraform and Ansible on AWS. The infrastructure is fully provisioned using Infrastructure as Code and configured automatically using Ansible.

Infrastructure Components

  • Custom VPC with public subnets across multiple Availability Zones
  • Internet Gateway and Route Tables
  • 2 EC2 instances deployed using Terraform
  • Application Load Balancer (ALB) for high availability
  • Security Groups allowing SSH and HTTP traffic
  • S3 bucket created via Terraform

Automation Tools Used

Terraform
Ansible
AWS EC2
AWS ALB
AWS VPC
Linux

Configuration Management

  • Apache/Nginx installed and configured using Ansible
  • HTML application deployed automatically
  • Dynamic Ansible inventory generated from Terraform outputs
  • Passwordless SSH authentication using EC2 key pairs

Why This Project Matters

This project reflects how infrastructure and configuration are handled in real production environments using automation instead of manual setup. It demonstrates best practices in scalability, repeatability, and security.
